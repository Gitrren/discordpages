# WGS bot legal pages

Static GitHub Pages site for the Terms of Service and Privacy Policy pages used by WGS Discord bots.

## Links

After publishing this folder with GitHub Pages, use these paths in the Discord Developer Portal:

- `/pokexbot/privacy/`
- `/pokexbot/terms/`
- `/thomas/privacy/`
- `/thomas/terms/`
- `/chopper/privacy/`
- `/chopper/terms/`

## Publish

1. Create a new GitHub repository, for example `bot-legal-pages`.
2. Upload/push this folder to the repository.
3. In GitHub, open `Settings -> Pages`.
4. Set source to `Deploy from a branch`, branch `main`, folder `/root`.
5. Use the generated GitHub Pages URLs in each Discord app.
